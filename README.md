This project is a Twitter Data Analysis Dashboard built using Streamlit. The main goal is to help users understand large collections of tweets in an easy, visual, and interactive way. Tweets are often unstructured and difficult to read one by one, so this dashboard provides tools to instantly explore key insights.

The application allows users to upload a CSV file of tweets and automatically performs:

1. Tweet Cleaning & Preprocessing

Removes URLs, mentions, punctuation, and stopwords

Converts text into a cleaner format for analysis

2. Word Frequency Analysis

Identifies the most common words used in the tweets

Helps users understand what topics are discussed the most

3. Word Cloud Generation

Creates a visual word cloud representing the most frequent words in the dataset

Gives a quick snapshot of the main themes and topics

4. Data Exploration

Displays uploaded tweets

Shows number of tweets, distribution, and patterns

Provides charts and simple visual analytics

5. Interactive UI

Clean navigation bar

Multiple pages for exploration

Simple interface suitable for beginners and analysts
